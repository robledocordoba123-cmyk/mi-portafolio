# Portafolio Personal de Manuela Córdoba

¡Bienvenido a mi portafolio! Este sitio web fue desarrollado como proyecto final usando **HTML5, CSS3 y Multimedia**. Aquí podrás conocer mi trabajo, mis proyectos y mi experiencia.

---

## Contenido del Portafolio

- **Inicio** (`index.html`): Presentación principal del portafolio.
- **Sobre mí** (`about.html`): Información personal y trayectoria.
- **Servicios** (`servicios.html`): Detalles de los servicios que ofrezco.
- **Multimedia** (`multimedia.html`): Galería de fotos y videos de mis proyectos.
- **Contacto** (`contacto.html`): Formulario y medios de contacto.
